# ai-chatbot
Developed a responsive AI chatbot web application using HTML, CSS, and JavaScript featuring interactive user messaging, smart automated responses, dark/light mode, chat history storage, typing animation, and responsive UI design.
IB AI Assistant

IB AI Assistant is a responsive AI chatbot project built using HTML, CSS, and JavaScript.

 Features
- Smart chatbot responses
- Dark and light mode
- Chat history storage
- Typing animation
- Real-time clock
- Responsive design
  
Technologies Used
- HTML5
- CSS3
- JavaScript
